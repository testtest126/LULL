# LULL — Roadmap

Horror by permission, first: uncanny, not traumatizing; dread over gore;
consent-first, every sensor opt-in, explained, and revocable (see
[`CLAUDE.md`](CLAUDE.md) §1). Nothing below is worth building if it can't
hold that line.

## Now
- Consolidate the ideation docs — THE MIRROR (PR #4) and THE IDOL / kintsugi (PR #7) — plus the SwiftGodot "stay native" decision (PR #6) into a single design-pillars doc.
- Pick the first mechanic to prototype: `THE EYE` is the spine.

## Next
- Build a vertical slice of `THE EYE` — the consent flow + `CameraGate` + the `Atmosphere` voices (Kafka / Beckett / Poe + Bulgakov, PR #5) as one playable ~5-minute sitting.

## Later
- The "haunt" Vapor server for cross-session persistence — which would earn chess-style security gating once it exists.
- THE MIRROR and THE IDOL as additional slices.
- App Store consent / review prep for a camera-driven horror app.
